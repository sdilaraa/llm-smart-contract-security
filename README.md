# LLM-Based Smart Contract Security Analysis

## Project Description
This project focuses on detecting vulnerabilities in smart contracts using Large Language Models (LLMs).

## Objective
- Identify potential security vulnerabilities  
- Analyze smart contract code  
- Apply LLM-based techniques  

## Project Structure
- `src/` → Python analysis code  
- `contracts/` → Smart contract samples  

## Technologies
- Python  
- Solidity  
- LLM / NLP  
- Blockchain concepts
