# LLM-Based Smart Contract Security Analysis

## Project Description
This project aims to analyze smart contracts and detect potential vulnerabilities using Large Language Models (LLMs).

## Objective
- Detect common vulnerabilities (reentrancy, overflow, access control)
- Analyze Solidity smart contracts
- Provide AI-based explanations

## Project Structure
- `src/` → Python analysis code  
- `contracts/` → Sample smart contracts  

## How It Works
1. A Solidity smart contract is provided as input  
2. The Python script analyzes the contract  
3. LLM-based logic is used to identify potential vulnerabilities  

## Technologies
- Python  
- Solidity  
- LLM / NLP
